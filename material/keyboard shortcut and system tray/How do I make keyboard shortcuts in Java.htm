    Do you use SWING?

    If you use SWING, this is the feature:

    JButton b = new JButton("Login");
    b.setMnemonic('L');

    Note that L is a character and the shortcut now is Alt+L but is obsolete.

    You can also use
    JButton b = new JButton("Login");
    b.setMnemonic(KeyEvent.VK_L);

    KeyEvent is class from library java.awt.event.KeyEvent;
